This tool is designed to test hdd/ssd and collect fio & iostat metrics, after which gnuplot is used to visualize them.

available scenario:
1. iops
2. bandwidth
3. latency
4. fixed iops
5. fixed bandwidth

available target:
1. single device
2. multile device

available fio rw mode:
1. read
2. write
3. randread
4. randwrite
